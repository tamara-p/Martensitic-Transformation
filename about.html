About the site

This website was created by Tamara Puskarevic during a four hour website building competition which was meant to help participants develop their coding skills. The prompt of the competition was to create a website that explained any concept. The topic of Martensitic Transformation was chosen because Tamara worked on a project involving the topic last summer, and created several animations depicting martensitic transformation with Michael Smith for professor Sangwoo Lee at Rensselaer Polytechnic Institute, which can be seen on the home page.
